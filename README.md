## Afromerica Entertainment website
Afromerica Entertainment's web platform, designed to support content management, event ticketing, artist showcasing, and administrative functions
---

### Development plan
    * Setup Project
    * Configure CMS
    * Design Database Schema
    * Setup Authentication
    * Build out Public Pages -- Home, About, Events, Artist, Contact
    * Implement CMS Integration
    * Create Booking System
    * Integrate Paystack Payment
    * Build Admin Authentication
    * Create Admin Dashboard
    * Implement CRUD Operations
    * Add analytics views
    * Test And Add Performance Optimization and Security Audit
    * Deploy to Production
    * Configure logging and monitoring 
    * Setup CI/CD
    * Final Testing

---

Built using *Next.js*, *Typescript*, *Tailwind CSS + Shadcnui Components*, *Zustand*, *Axios*, *Prisma*, 
*Sanity.io*, *PostgreSQL*, *Redis*
